# terraform-ec2-basic ☁️

Launch an AWS EC2 instance using Terraform Infrastructure as Code.

### Tech Stack
AWS, Terraform, IaC

### What This Does
Creates a `t2.micro` EC2 instance in Mumbai region `ap-south-1` using Terraform.

### Prerequisites
- AWS Account with IAM keys configured
- Terraform installed locally

### How to Run
```bash
terraform init
terraform plan
terraform apply
